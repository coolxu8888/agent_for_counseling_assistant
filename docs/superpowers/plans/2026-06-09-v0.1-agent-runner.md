# v0.1 Agent Runner Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a local command-line agent runner that accepts real user input for W1/W2/W3, assembles a RAG-backed prompt package, calls DeepSeek, and saves raw/clean/safety artifacts under `agent-runs/`.

**Architecture:** Add a focused `scripts/run_agent.py` module with workflow normalization, input validation, deterministic RAG selection, prompt assembly, run-folder persistence, DeepSeek API reuse, and safety checks. Keep eval runner behavior unchanged; the agent runner can reuse existing DeepSeek config/HTTP helpers and clean/rubric functions where practical.

**Tech Stack:** Python standard library (`argparse`, `json`, `pathlib`, `datetime`, `tempfile`, `unittest`), existing DeepSeek runner helpers, PowerShell wrapper, existing RAG markdown files.

Create `scripts/test_run_agent.py` with tests for workflow aliases and input validation:

```python
import unittest

from run_agent import (
    AgentInputError,
    normalize_workflow,
    read_user_input,
)


class RunAgentTest(unittest.TestCase):
    def test_normalize_workflow_accepts_aliases(self):
        self.assertEqual(normalize_workflow("W1").workflow_id, "W1")
        self.assertEqual(normalize_workflow("intake").workflow_id, "W1")
        self.assertEqual(normalize_workflow("case").workflow_id, "W2")
        self.assertEqual(normalize_workflow("session").workflow_id, "W3")

    def test_normalize_workflow_rejects_unknown_alias(self):
        with self.assertRaisesRegex(AgentInputError, "W1"):
            normalize_workflow("unknown")

    def test_read_user_input_requires_exactly_one_source(self):
        with self.assertRaisesRegex(AgentInputError, "exactly one"):
            read_user_input(None, None)
        with self.assertRaisesRegex(AgentInputError, "exactly one"):
            read_user_input("inline", "notes.md")

    def test_read_user_input_accepts_inline_text(self):
        source, text = read_user_input("  来访者材料  ", None)

        self.assertEqual(source, "inline")
        self.assertEqual(text, "来访者材料")
```
