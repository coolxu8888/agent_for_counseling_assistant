# 工程证据与审计记录

本目录保存市场验证版的来源追溯、安全边界解释、评测依据和关键产品决策。这里的文件不参与 Web 运行时检索，也不会被拼装进模型 prompt；文件名中的 `v0.1` 是历史版本标识，不代表当前产品阶段。

## 目录

- `sources/`：来源卡、拒绝来源和来源审核指南，用于追溯 RAG 规则的出处与取舍。
- `evaluations/`：历史测试用例、风险 rubric、人工评测说明、回归记录和评测摘要。
- `decisions/`：产品范围和关键安全决策的历史记录。
- `governance/`：RAG 建库、审核、切块和 metadata 治理规则。

## 使用边界

- 这些材料是工程证据，不是临床指南或自动决策规则。
- 历史评测结果只证明对应提交和测试条件下的表现，不代表当前模型始终满足同一结果。
- 来源地域、时效性和适用范围仍需人工复核；境外专业材料不能替代中国本地法律、伦理和机构流程。
- 历史文件中的原路径引用用于记录当时仓库结构，移动到本目录后可能不再是可执行路径。

## 更新要求

涉及风险边界、隐私规则、RAG 来源或评测标准的变更，应同步更新本目录中的对应证据，记录来源、审核结论和回归结果。不得以“运行时未读取”为由删除唯一审计依据。
