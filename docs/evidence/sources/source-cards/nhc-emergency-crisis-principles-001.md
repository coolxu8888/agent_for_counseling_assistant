# 摘要

《紧急心理危机干预指导原则》由卫生部办公厅印发，用于指导重大灾害等突发事件后的紧急心理危机干预工作。该资料强调心理危机干预应在整体救援或应急工作部署下开展，应以科学态度对待危机干预，识别重点人群，制定干预方案，并在必要时向相关负责人报告和提供建议。对咨询师助理 Agent v0.1 来说，它比热线指南更接近“危机干预原则”资料，但仍然属于突发事件/灾后场景，不是普通心理咨询个案的完整 SOP。

# 可用内容

- 心理危机干预应结合整体应急或救援部署，避免孤立开展。
- 应以科学态度看待心理危机干预，不将其视为万能工具。
- 应根据掌握的信息识别重点人群和紧急心理事件苗头。
- 应制定干预方案，并根据工作进展调整干预重点。
- 可用于强化 v0.1 的边界：识别风险、提示进一步专业评估、必要时按机构流程报告/转介。

# 不适用或需谨慎内容

- 该资料主要面向突发事件和灾后紧急心理危机干预，不是普通心理咨询中心的完整危机处置 SOP。
- 不应让 Agent 直接执行危机干预流程或调度外部资源。
- 不应据此自动判定风险等级或替代精神卫生专业人员判断。
- 若用于学校、机构、热线或医疗场景，需要结合该场景自己的制度。
