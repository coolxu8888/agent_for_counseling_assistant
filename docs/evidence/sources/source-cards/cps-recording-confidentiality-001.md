# 摘要

本资料卡基于《中国心理学会临床与咨询心理学工作伦理守则（第二版）》中与专业资料、个案记录、保密、录音录像、案例报告和去识别化相关的内容，用于替代原 HHS/HIPAA 语境下的心理治疗笔记资料卡。该守则明确将个案记录、测验资料、信件、录音、录像和其他资料视为心理师工作中的专业信息，并要求心理师在创建、使用、保存、传递和处理这些资料时遵守法律和专业伦理规范。对 v0.1 来说，它更适合作为中文语境下 session 记录、个案信息整理和咨询记录边界的资料来源。

# 可用内容

- 专业资料包括个案记录、测验资料、信件、录音、录像等。
- 创建、使用、保存、传递和处理专业资料时，应遵守相关法律和专业伦理规范。
- 与服务相关的录音、录像或演示，原则上需要服务对象同意。
- 发表案例报告或教学科研使用相关材料时，应保护服务对象身份，避免可识别信息暴露。
- 可用于强化 v0.1 中“记录资料属于敏感专业信息”“只保留最小必要信息”“不扩散无关隐私”“输出中避免可识别细节”的规则。

# 不适用或需谨慎内容

- 该守则是伦理规范，不是具体电子病历、医疗病历或机构档案管理制度。
- 不应把伦理原则转写成具体法律意见。
- 不应替代机构关于记录保存年限、数据安全、访问权限和档案管理的具体制度。
- 不应把“案例去识别化”理解为可以自由公开个案材料；仍需审查再识别风险和授权范围。
