# 摘要

APA Record Keeping Guidelines 为心理学家提供专业记录保存的决策框架，强调记录内容会因服务目的、设置和语境而变化，并提醒从业者熟悉所在专业语境和司法辖区的法律与伦理要求。对 v0.1 来说，这份资料适合支持“记录应服务于后续服务、进展追踪、法律/伦理程序和连续照护”的思想。当前 URL 已指向 APA 官方域名，可作为候选官方来源继续审核。

# 可用内容

- 记录保存需要考虑法律、伦理和专业语境。
- 记录有助于服务延续、治疗计划、服务过程和进展记录。
- 记录详细程度应依据服务目的、设置和语境而定。
- 可支持 Workflow 2 和 Workflow 3 的记录原则。

# 不适用或需谨慎内容

- 该资料发布于 2007 年，需确认是否仍为 APA 当前建议或是否已有更新版本。
- 该资料属于美国专业与法律语境，不应直接套用于其他地区。
- 不应作为唯一记录保存规范。
- 不应用于给出法律意见。
