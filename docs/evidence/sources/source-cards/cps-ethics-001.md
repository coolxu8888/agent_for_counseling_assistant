# 摘要

《中国心理学会临床与咨询心理学工作伦理守则（第二版）》由中国心理学会发布，刊载于《心理学报》2018 年第 50 卷第 11 期。该守则由中国心理学会授权临床心理学注册工作委员会在 2007 年第一版基础上修订，旨在说明临床与咨询心理学服务工作的教育性、科学性与专业性，帮助心理师、寻求专业服务者和公众理解专业伦理核心理念与专业责任。对 v0.1 来说，它比原先的英国心理学会伦理资料更适合作为中国大陆语境下的伦理底座，用于约束 Agent 在知情同意、保密、专业关系、胜任力、风险边界和记录表达中的基本规则。

# 可用内容

- 作为中国大陆语境下临床与咨询心理学工作的伦理参考。
- 支持 v0.1 中“Agent 不替代咨询师专业判断、不输出确定性诊断、不自动进行危机处置”的边界。
- 可用于初访表中的知情同意、保密与保密例外、咨询关系说明、专业服务边界。
- 可用于个案信息整理和 session 记录中的事实/推测区分、敏感信息最小必要、专业表达约束。
- 可支持高风险信息出现时提醒咨询师进行进一步专业评估，并按机构流程处理或转介。

# 不适用或需谨慎内容

- 该守则是伦理规范，不是具体咨询记录模板、风险评估量表或危机干预 SOP。
- 不应将守则内容转写为自动风险分级规则。
- 不应作为诊断、治疗方案或自动危机处置依据。
- 若产品服务特定机构、学校、医疗或企业场景，仍需结合该机构制度、当地法律法规和督导要求。
- 若后续产品面向非中国大陆地区，需要另行补充当地伦理与法律规范。
