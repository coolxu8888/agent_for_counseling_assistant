# 摘要

SAMHSA SAFE-T 是面向临床人员的自杀评估五步框架，包含识别风险因素、识别保护因素、进行自杀询问、确定风险水平与干预、记录评估与随访计划。对 v0.1 来说，它适合支持初访风险筛查、个案整理中的风险信号识别，以及 session 记录中的风险变化记录。但 v0.1 仍不自动判断最终风险等级或替代咨询师临床决策。

# 可用内容

- 风险因素与保护因素都需要识别。
- 自杀询问应包括想法、计划、行为、方法和意图。
- 风险评估过程需要记录评估依据、干预和随访计划。
- 可用于 v0.1 的风险栏目设计和记录检查。

# 不适用或需谨慎内容

- SAFE-T 是美国公共卫生语境下的工具。
- v0.1 不应自动执行“确定风险水平与干预”这一临床决策步骤。
- 不应将 SAFE-T 作为唯一风险评估来源。
