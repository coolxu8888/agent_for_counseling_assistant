# 摘要

SAMHSA 资源目录中的 C-SSRS 条目介绍了 Columbia-Suicide Severity Rating Scale 是一个可快速使用的自杀意念与行为筛查/评估问卷，适用于多种场景和年龄群体。对 v0.1 来说，它可以作为“自杀风险询问应覆盖意念、行为和严重程度”的候选参考，但不应在未经授权和人工审核前直接复制量表条目，也不应让 Agent 自动判定风险等级。

# 可用内容

- 可作为自杀风险询问主题的候选来源。
- 可提醒初访和 session 记录中关注自杀意念与行为。
- 可与 SAFE-T 一起作为风险筛查资料卡参考。

# 不适用或需谨慎内容

- 不应直接复制 C-SSRS 问卷或评分规则，除非确认授权和使用条款。
- 不应让 v0.1 自动完成风险分级。
- 不应作为唯一风险评估来源。
