# 摘要

该 BACP Good Practice in Action 资源专门讨论咨询专业中的保密与记录保存。它强调记录保存与保密需要结合伦理、法律和实践语境来思考，并将记录理解为纸质、电子或其他形式的文件。对 v0.1 来说，这份资料适合支持“咨询记录需要准确、适当、相关”“录音/录像需要知情同意”“记录保存涉及数据保护”等规则。

# 可用内容

- 记录不仅包括纸质文档，也包括电子和其他形式记录。
- 咨询记录需要与保密、法律和伦理一起考虑。
- 音频或视频记录 session 时需要考虑知情同意。
- 可为 Workflow 3 的 session 记录、Workflow 2 的个案信息整理提供边界。
