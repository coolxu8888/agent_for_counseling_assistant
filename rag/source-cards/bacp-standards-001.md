# 摘要

BACP Ethical Framework 的“Working to Professional Standards”页面包含与专业胜任力、持续学习、转介、协作和记录保存相关的要求。对 v0.1 最有用的是：咨询工作需要保持专业胜任力；必要时应咨询、寻求第二意见或转介；记录应准确、相关，并限于服务所需。这些原则可以直接支持个案信息整理和 session 记录中的“最小必要信息”“不编造”“提醒转介”等规则。

# 可用内容

- 记录应准确、相关，并限于服务类型所必需。
- 当专业标准需要外部专业意见、第二意见或转介时，应在保护保密与数据的前提下进行。
- 服务无论线上或线下，都应符合基本专业标准。
- 可用于强化 v0.1 的记录边界和转介提醒。

# 不适用或需谨慎内容

- BACP 是英国咨询与心理治疗专业协会，适用语境为 UK。
- 不应把 BACP 的记录原则直接等同于所有地区的法定要求。
- 不提供具体 SOAP、DAP、BIRP 模板。
