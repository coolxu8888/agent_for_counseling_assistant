Use this chunk in Workflow 6 when the counselor asks for a counseling roadmap, phased plan, or multi-session direction for a de-identified case.

Output frame:

- Start with a brief overview that names the selected framework and states that the roadmap is revisable.
- Organize the roadmap into phases rather than promising a fixed number of sessions.
- For each phase, describe the counselor's working focus, what hypotheses still need verification, and what markers should be monitored.
- Offer session focus options rather than one mandatory sequence.
- Keep risk monitoring checkpoints visible across phases.
- Add collaboration or referral reminders only as conditional prompts for counselor judgment.
- Add missing-information prompts when the case material does not support a fuller roadmap.
- End with explicit do-not-do boundaries.

Do not turn the roadmap into a diagnosis, guaranteed outcome, final risk judgment, crisis decision, or rigid treatment prescription. Do not promise a fixed 8-session or 12-session course unless the user explicitly provided a program structure.
