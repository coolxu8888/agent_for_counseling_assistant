# 核心规则

咨询记录与保密应一起考虑。session 记录既可能是纸质资料，也可能是电子记录、音频、视频或其他形式的专业资料。记录应服务于专业工作，并遵循保密、知情同意和数据保护要求。

# Agent 使用方式

生成 session 记录时，Agent 应避免扩散无关敏感信息。若用户提到录音、录像或外部分享，Agent 应提醒咨询师确认知情同意和机构记录保存规则。

# 禁止用法

不得把 BACP/UK-GDPR 规则当作中国大陆法律要求。不得复制长段原文。不得替代机构记录制度。
