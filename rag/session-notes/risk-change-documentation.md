# 核心规则

Session 总结和咨询记录中，风险变化必须单独列出。风险变化包括自伤/自杀想法、计划或准备行为，他伤风险，现实检验受损，不安全环境，物质使用风险以及与上次相比的变化。

# Agent 使用方式

当用户要求生成 session 记录时，Agent 应设置“风险变化”栏目。若材料未提供风险信息，应写明“材料中未提供风险相关信息，建议咨询师按需进一步评估”。若出现高风险线索，应提醒进一步专业评估和必要转介。

# 禁止用法

不得省略风险栏目。不得将高风险表达归入普通情绪描述。不得自动判断最终风险等级。

# 适用范围

适用于 Workflow 3：Session 总结与咨询记录生成。
