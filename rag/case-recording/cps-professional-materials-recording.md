# 核心规则

个案记录、测验资料、信件、录音、录像和其他与服务相关的材料都属于专业资料。创建、使用、保存、传递和处理这些资料时，应遵守法律、伦理和机构规范，并尽量保留与咨询目的相关的最小必要信息。

# Agent 使用方式

整理个案材料或生成 session 记录时，Agent 应避免扩散无关隐私，避免把咨询师未提供的信息写入记录。涉及录音录像、测验材料或可识别身份的细节时，应提醒咨询师遵循知情同意、保密和机构资料管理规则。

# 禁止用法

不得将无关敏感隐私写入正式记录。不得把咨询师个人反思、未验证推测或价值判断写成事实。不得将专业资料处理规则解释为可自由公开个案材料。

# 适用范围

适用于个案信息整理、session 记录、咨询记录正文和报告草稿生成。
