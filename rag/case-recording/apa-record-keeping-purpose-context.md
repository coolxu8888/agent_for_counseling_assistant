# 核心规则

专业记录应服务于持续照护、治疗计划、服务过程、进展记录和必要的专业沟通。记录内容和详细程度应结合服务目的、服务场景和专业语境决定。

# Agent 使用方式

当生成个案摘要或 session 记录时，Agent 应让记录服务于后续咨询工作：保留主诉、关键事件、咨询师干预、来访者反应、风险变化、进展和下次计划。

# 禁止用法

不得把 APA 记录指南当作中国大陆法律要求。不得将记录写成诊断证明。不得为了连续性而添加用户未提供的信息。
