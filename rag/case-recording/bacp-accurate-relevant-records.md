# 核心规则

咨询相关记录应准确、相关，并限于服务所需。记录应帮助服务延续、专业反思和必要沟通，而不是收集或扩散无关信息。

# Agent 使用方式

整理个案或 session 记录时，Agent 应优先记录用户提供的事实、主诉、观察、干预、反应、风险和后续计划。对无关隐私、价值判断或未经验证推测，应避免写入正式记录。

# 禁止用法

不得编造咨询师未提供的信息。不得把推测写成事实。不得为了“完整”记录过多无关隐私。

# 适用范围

该 chunk 来自英国 BACP 资料，只作为记录原则补充参考。中国大陆使用时应结合 CPS 伦理和个人信息保护法。
