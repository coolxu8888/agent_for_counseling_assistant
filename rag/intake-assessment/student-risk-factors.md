# 核心规则

学生心理健康场景中，初访和个案整理应关注学业就业压力、经济困难、情感危机、家庭变故、校园欺凌、实习实践环境变化、社会支持不足等风险因素。这些因素可作为信息收集和信息缺口检查维度。

# Agent 使用方式

当来访者为学生或咨询场景为学校/高校时，Agent 可在初访表或补充访谈问题中加入学生场景相关风险因素，并提醒咨询师关注学校、家庭、医疗和社会支持系统。

# 禁止用法

不得将学校场景规则套用于所有成人个体咨询。不得基于单个风险因素直接判定危机等级。

# 适用范围

适用于学生心理健康、学校咨询、高校咨询和未成年人/青年学生相关场景。
