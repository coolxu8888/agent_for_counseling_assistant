# Core rule

Use psychodynamic next-session planning to shape one upcoming session around hypotheses about recurring relationship patterns, affect shifts, defenses, and meaning-making that can be explored collaboratively. Keep interpretations tentative and anchored in the current material.

# Agent use

Use this chunk only in Workflow 5 when the counselor asks for a psychodynamic next-session plan. Focus on one session goal, likely themes to observe, questions that invite reflection, and bounded risk-monitoring reminders without expanding into a multi-session formulation.

# Forbidden use

Do not present unconscious motives, developmental explanations, or transference interpretations as fact. Do not turn the session plan into a long-term roadmap or use psychodynamic language to imply diagnosis.

# Scope

Applies to counselor-facing planning support for one upcoming session only. It is suitable for de-identified supervision and preparation, not direct client advice.
