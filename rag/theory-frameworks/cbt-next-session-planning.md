# Core rule

Use CBT next-session planning to focus one upcoming counseling session on a clear, testable piece of the client's cognitive-emotional-behavioral cycle. Keep the plan bounded to one session and tie interventions to known material rather than speculative schema work.

# Agent use

Use this chunk only in Workflow 5 when the counselor asks for a CBT next-session plan. Prioritize one session goal, key cycle links to explore, candidate questions, and optional between-session tasks that remain subject to counselor judgment.

# Forbidden use

Do not turn the next session into a full treatment protocol, guaranteed homework assignment, or diagnosis. Do not prescribe exposure, restructuring, or behavioral experiments as mandatory when the counselor has not established readiness, safety, or fit.

# Scope

Applies to counselor-facing planning support for one upcoming session only. It is suitable for de-identified supervision and preparation, not direct client advice.
