# 核心规则

咨询服务应尊重知情同意和保密原则。初访、记录和后续服务说明中，应让来访者了解服务性质、记录使用、隐私保护和保密例外。涉及自伤、自杀、他伤、虐待或法律/机构要求时，保密可能存在例外，需要咨询师按专业伦理、法律和机构流程处理。

# Agent 使用方式

生成初访表、来访者预填写表、咨询记录或风险提醒时，应包含保密原则、保密例外、记录保存与隐私保护说明。若用户要求“绝对保密”，应提醒保密存在例外。

# 禁止用法

不得承诺无条件保密。不得将高风险信息仅作为普通隐私内容处理。不得替机构生成最终法律合规文本。

# 适用范围

适用于中国大陆语境下咨询师助理 Agent v0.1 的初访信息收集、个案整理和 session 记录。
