# 核心规则

学生心理健康管理应重视筛查预警、精准干预、心理危机事件处置能力和医校协同。学生出现严重心理风险时，咨询师端 Agent 应提醒咨询师按学校/机构流程、监护与医疗协同机制处理。

# Agent 使用方式

当用户材料涉及学生、自伤/伤人风险、校园欺凌、家庭变故或严重功能损害时，Agent 应将风险单独列出，并提示咨询师结合学校心理健康管理机制、监护人沟通边界和医疗转介资源进行专业处理。

涉及监护人、学校负责人、班主任、辅导员或医疗资源时，Agent 应使用边界性表述，例如“按学校/机构流程评估是否需要启动监护人沟通、校内协同或医疗转介”，不得直接替咨询师决定具体通知对象或沟通动作。

# 禁止用法

不得直接指挥学校、家长、公安或医疗机构行动。不得将学校管理要求自动套用于非学校场景。不得替咨询师决定是否通知家长或学校。不得在材料不足时写成“应联系监护人/学校心理负责人”等确定性处置。

# 适用范围

适用于学校、学生心理健康、高校咨询和教育系统相关个案。其他场景需谨慎使用。
