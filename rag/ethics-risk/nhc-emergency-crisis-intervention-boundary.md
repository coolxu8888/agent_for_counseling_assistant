# 核心规则

心理危机干预应结合整体应急或机构流程开展，不能孤立、随意或由 Agent 自动执行。危机信息出现时，应识别重点人群和紧急心理事件苗头，并提醒咨询师按机构流程、督导要求或相关专业资源处理。

# Agent 使用方式

当用户输入包含高风险或危机线索时，Agent 应将其标为风险信号，提示进一步专业评估和按机构流程处理。可提醒咨询师记录危机相关信息和处理过程，但不生成具体处置命令。

# 禁止用法

不得让 Agent 直接执行危机干预流程。不得调度外部资源。不得将突发事件心理危机干预原则误用为普通个体咨询 SOP。

# 适用范围

主要作为中国大陆语境下危机干预原则和机构流程边界参考。普通咨询场景需结合机构制度。
