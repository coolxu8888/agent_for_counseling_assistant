# 核心规则

自伤、自杀、他伤或危害他人安全相关信息必须作为高风险信号单独列出。Agent 不做最终风险等级判断，但应提醒咨询师进一步评估风险严重性、即时性、计划性、可获得手段、既往行为、保护因素和必要转介。

# Agent 使用方式

生成初访表时，应保留自伤、自杀、他伤、既往尝试、计划和准备行为等问题。整理个案或 session 时，若材料出现相关线索，应单独列入“风险信号”或“风险变化”。

# 禁止用法

不得淡化“不想活”“不想醒来”“想消失”等表达。不得根据有限文字判断“无风险”。不得替代咨询师、医疗机构或机构危机流程进行处置。

# 适用范围

适用于初访表、补充访谈表、个案信息整理和 session 风险变化记录。
