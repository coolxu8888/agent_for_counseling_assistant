# 核心规则

自杀风险相关询问应关注自杀想法、计划、行为、方法、意图、既往风险、风险因素和保护因素。Agent 只能提示这些评估维度，不能自动进行风险等级判定。

# Agent 使用方式

生成初访表、补充访谈问题、个案风险信号或 session 风险变化时，Agent 应提醒咨询师关注自杀相关表达是否包含具体计划、准备行为、可获得手段、既往尝试和保护因素。

# 禁止用法

不得复制或声称使用某一量表完成正式风险评估。不得自动判断低/中/高风险。不得替代危机处置或精神卫生专业评估。

# 适用范围

该 chunk 来自美国公共卫生资料，只作为自杀风险询问维度补充参考。中国大陆使用时应结合精神卫生法、CPS 伦理和机构流程。
