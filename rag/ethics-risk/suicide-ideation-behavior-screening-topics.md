# 核心规则

自杀风险筛查应区分自杀意念与自杀行为，并关注严重程度、近期变化和既往行为。该 chunk 只提供筛查主题，不复制量表题项或评分规则。

# Agent 使用方式

当材料中出现“不想活”“想消失”“不想醒来”等表达时，Agent 应提示咨询师进一步了解是否存在自杀意念、计划、准备行为、既往尝试、行为冲动和保护因素。

# 禁止用法

不得复制 C-SSRS 量表题项或评分规则。不得声称完成正式量表评估。不得自动判断风险等级。

# 适用范围

作为自杀风险筛查主题补充参考。正式评估需要受训专业人员和合适工具。
