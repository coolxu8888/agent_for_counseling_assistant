### **咨询路线图（初步框架，可调整）**

**概述：**
本路线图采用一个渐进式的、以信息收集与稳定化为导向的框架。路线图分为三个主要阶段，每个阶段都有明确的工作重点、待验证假设和监测指标。**请注意，该路线图不预设固定咨询次数，将根据来访者的进展和风险变化进行灵活调整。**

### **第一阶段：初始评估与稳定化（包含下一次咨询）**

**核心目标：** 建立安全稳定的咨询关系，完成基础风险评估，澄清核心困扰，并填补信息缺口。

| 阶段重点 | 具体工作方向 |
| :--- | :--- |
| **下一次咨询焦点** | 1. **深度探索主诉与困扰**：基于session note中“不想醒来”等表述，温和地、非评判地探索其背后的情绪、想法和具体情境。避免推测，集中于来访者愿意分享的主观体验。<br>2. **风险评估与澄清**：在安全、建立了初步信任的前提下，系统性地评估当前的自杀意念细节（如频率、强度、触发因素），并明确区分其与自伤或他伤意向。这是首要且持续的工作。<br>3. **信息缺口填充**：根据session note中未提及的内容，有选择地询问以下关键领域，以建立更全面的背景：<br>   - **身体健康**：近期的身体状况、睡眠、饮食、精力水平变化（生物维度）。<br>   - **社会支持**：来访者生活中“可以联系的朋友”之外，是否还有其他人（家人、同事等）可以提供支持？与这些人的关系质量如何？（社会维度）<br>   - **前因与触发事件**：当前困扰开始或加剧的时间点，有无明显诱因？（心理社会维度）<br>4. **建立咨询框架**：与来访者确认咨询目标、频率、时长、保密及其例外原则，并明确告知咨询不是危机热线服务。 |
| **待验证的初步假设** | 1. 来访者的“不想醒来”可能指向严重的情绪困扰（如抑郁情绪），但缺乏足够信息进行推断。<br>2. 社会支持系统可能存在不足，或来访者感到难以利用现有支持（如“很久没有告诉朋友自己的状态”）——此假设源于来访者描述，需进一步评估。 |
| **需持续监测的指标** | - **风险信号**：自杀意念的任何变化（如出现具体计划、准备行为或企图）。<br>   **注意**：需精确区分自杀意念、自伤、他伤。当前材料仅支持自杀相关表达，自伤/他伤风险标记为“材料未提及”。<br>- **情绪状态**：情绪是否出现剧烈波动、极端的绝望感或躁动。<br>- **功能水平**：工作/学习、日常自理和基本社交能力是否出现明显下降。 |
| **条件性提醒** | - **转介/协同提醒**：如果评估发现来访者存在中度以上自杀风险（如出现具体计划或强烈意向），请咨询师判断是否需要：<br>  1. 启动校内/机构危机干预流程。<br>  2. 联合精神科医生进行药物评估。<br>  3. 通知紧急联系人（需遵循保密例外原则）。<br>  **该提醒不替代咨询师的最终专业判断。** |

### **第二阶段：深入探索与理解（后续2-4次咨询）**

**核心目标：** 在风险可控和咨询关系稳固的基础上，深入探索困扰的根源、模式和影响。

| 阶段重点 | 具体工作方向 |
| :--- | :--- |
| **工作焦点** | 1. **概念化邀请**：在积累了足够数据后，可以引入一个或多个理论视角（如CBT的认知三角，精神动力的防御模式，或依恋理论）来帮助来访者理解自身困扰，但这**不**是去完成一个诊断。<br>2. **模式识别**：帮助来访者识别其情绪、想法和行为之间的模式，以及这些模式如何在其人际关系和生活中反复出现。<br>3. **脆弱性与资源**：探讨来访者在哪些情境下最脆弱（如压力、孤单、失败感），又拥有哪些内在或外在的资源来应对。 |
| **待验证的初步假设** | 1. 来访者特定的认知偏差（如“我毫无价值”、“未来没有希望”）。<br>2. 来访者的关系模式（如回避型依恋、过度依赖）。<br>3. 来访者的核心情绪（如羞耻感、无望感、愤怒）。<br> *以上所有假设均需基于后续session中来访者提供的信息进行验证。* |
| **需持续监测的指标** | - **风险信号**：持续监测，尤其注意当探索到核心冲突时，自杀风险是否升高。<br>- **咨询关系**：来访者是否感到被理解、信任，并愿意披露更多。<br>- **改变动机**：来访者表达改变的意愿和投入度。 |

### **第三阶段：巩固与行动（中长期方向）**

**核心目标：** 在理解的基础上，共同发展适应性应对策略，并巩固改变。

| 阶段重点 | 具体工作方向 |
| :--- | :--- |
| **工作焦点** | 1. **技能训练与策略发展**：根据来访者和咨询师共同的取向选择，发展具体技能（如情绪调节、人际沟通、认知重构）。<br>2. **行为实验与行动计划**：在低风险场景下，鼓励来访者尝试新的行为，以验证旧有信念或学习新应对方式。<br>3. **预防复发与巩固**：总结学到的经验，识别未来可能遇到的挑战，并制定相应的预防计划。 |
| **待验证的假设** | 来访者能够发展出替代自杀意念、寻求有效支持的能力。 |
| **需持续监测的指标** | - **风险信号**：即使好转，也应有计划地监测风险复发迹象。<br>- **改变泛化**：改变是否出现在咨询室外的生活中。<br>- **自主性**：来访者是否变得更有能力独立应对。 |
| **条件性提醒** | - **合作/转介提醒**：如果来访者需要长期、深度的治疗或药物管理，咨询师应考虑转介或与精神科协作。 |

**重要边界说明（本路线图不会做以下事情）：**

*   **不进行诊断：** 本路线图不给出任何确定性诊断。
*   **不替代危机处置：** 任何风险等级的判断，都以“候选等级”或“材料支持的倾向”形式呈现，且最终风险等级和危机处置决定，完全由咨询师结合面询、机构流程和专业判断做出。
*   **不承诺效果：** 本路线图是工作框架，不承诺治疗结果。
*   **不固定咨询次数：** 路线图以“阶段”而非“固定次数”来规划。
*   **不进行流派概念化：** 在v0.1阶段，本路线图基于通用信息收集框架，不强制使用任何特定流派理论。
