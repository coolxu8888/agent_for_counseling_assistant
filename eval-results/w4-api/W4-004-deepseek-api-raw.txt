*   **触发情境：** 来访者A报告，在最近一次小组汇报（工作场景）中，被一位同事当众提出了尖锐的质疑。来访者A当时未能立即回应，并感到“脸发烫，心跳加速”。
*   **主诉：** 来访者A描述此后两天都反复回想该场景，并感到“羞愧”和“焦虑”，担心同事会认为“我能力不行”。这影响了其准备下一项工作的专注度。
*   **行为模式：** 来访者A提到，为了避免类似情况，他暂时停止参加需要主动发言的线上讨论。

#### 二、CBT工作假设与核心循环

以下是对案例的初步理解，属于推测与工作假设，需要进一步验证。

1.  **核心自动思维：** 在触发情境下，可能出现的自动思维是：“我搞砸了”、“他们一定觉得我很没用。”
2.  **信念层面：** 这可能与一个潜在的中间信念有关——“我必须表现得完美无缺，才能被大家认可（或不被否定）”。该信念成为一种应对策略，使来访者对任何质疑都非常敏感。
3.  **情绪与身体反应：** 自动思维引发了“羞愧”和“焦虑”情绪，同时伴有脸红、心跳加速（身体反应）。
4.  **行为与维持循环：**
    *   **回避行为：** 停止参加线上讨论（安全/回避行为）。
    *   **维持机制：** 回避行为短期缓解了焦虑（负强化），但从长期看，它阻止了来访者获得“即使不完美，大家也可能不会因此否定我”的纠正性证据。同时，反复的思维反刍（不断回想那个场景）维持了负面的认知和情绪，形成了一个**认知-情绪-行为**的维持循环。
