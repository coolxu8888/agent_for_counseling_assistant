# 危机等级辅助判断 Rubric v0.1

## 来源

本 rubric 来自用户提供的训练数据文件：

```text
eval-data\agent_eval_training_data.20260531.xlsx
```

对应项目内评测数据：

```text
eval-data/crisis-level-cases.v0.1.json
```

其中：

- `Sheet1` 存放题目与标答。
- `Sheet2` 存放等级、标准、定义和备注。

## 使用边界

本文件用于咨询师助理 Agent 的训练、评测和督导辅助场景，帮助模型学习如何根据材料识别危机等级相关线索。

它不是临床诊断工具，不是正式风险评估量表，也不能替代咨询师、精神科医生、学校/机构危机流程或紧急处置。

当用于真实个案辅助时，Agent 只能输出：

- 候选危机等级；
- 支持该候选等级的材料依据；
- 不满足更高等级的原因；
- 需要进一步询问的信息；
- 按机构流程进一步评估或转介的提醒。

Agent 不得把候选等级写成最终风险等级。

## 等级框架

### 一级：被动式自杀意念

Sheet2 标准：

```text
被动式自杀意念
```

Sheet2 定义：

```text
指个体想在外力或者偶然性的作用下人意料地结其生命而非主动终止其生命的念头，是在没有明确目标时发生的一种心理反应；
```

辅助识别线索：

- 来访者表达希望在外力、偶然事件或非主动行为中死亡；
- 没有主动结束生命的明确主观意愿；
- 没有明确自杀计划或资源准备；
- 可能有短暂情绪崩溃、痛苦表达或“要是发生意外就好了”等表达。

输出重点：

- 标注为被动式自杀意念；
- 不淡化风险；
- 继续询问是否存在主动求死想法、计划、手段和保护因素。

### 二级：主动式自杀意念，无计划或计划模糊

Sheet2 标准：

```text
主动式自杀意念，无计划/模糊计划
```

Sheet2 定义：

```text
指个体期望通过其主动来终结其生命的念头，是由个人对自杀问题的态度引起的；
```

辅助识别线索：

- 来访者表达主动结束生命的想法；
- 有“想死”“不如死了”“活着没意思”等主动求死表达；
- 没有形成具体、可执行的自杀计划；
- 没有准备实施自杀所需资源，或资源/方式明显不可执行；
- 可能对方式、地点、时间、手段没有明确安排。

输出重点：

- 标注主动式自杀意念；
- 说明目前材料未显示可执行计划或资源；
- 继续评估计划是否具体化、资源是否可获得、意向是否增强。

### 三级：主动式自杀意念，有计划有资源，无实施意向

Sheet2 标准：

```text
主动式自杀意念，有计划有资源，无实施意向
```

Sheet2 定义：

```text
完成自杀行为所需要的前置客观条件，以及主动想自杀行为发生的时间、地点、方式等，要包含自杀计划及自杀资源，自杀资源较难获取的情况下（例如管制化学药品或者器械），自杀资源大于自杀计划，自杀资源较容易获取（例如小刀，跳楼等），自杀计划大于自杀资源
```

Sheet2 备注：

```text
三四级判断：当事人是否有既往自杀历史且是否有尝试自杀的经验；举例：割腕的程度；二三级判断：有自杀资源但该资源难获得，自杀计划即使模糊也能算三级；反之自杀资源容易获得，自杀计划就要具体才能算三级
```

辅助识别线索：

- 来访者有主动结束生命的想法；
- 已经出现较明确的自杀方式、地点、时间或资源；
- 具备实施所需的客观条件或资源；
- 但材料显示尚未形成明确执行决心，或存在重要保护因素使其未决定实施。

输出重点：

- 区分“有计划/资源”和“有实施意向”；
- 说明哪些材料支持计划和资源已经具备；
- 说明哪些材料显示执行意向尚不明确或被保护因素阻断；
- 提醒持续评估意向变化和资源可及性。

### 四级：主动式自杀意念，有计划有资源，有实施意向，无尝试

Sheet2 标准：

```text
主动式自杀意念，有计划有资源，有实施意向，无尝试
```

Sheet2 定义：

```text
个体执行自杀计划的决心
```

Sheet2 备注：

```text
根据主角的行为看我们要以四级风险（需关注）对待，还是以五级风险（需高度关注）对待
```

辅助识别线索：

- 来访者有主动结束生命的想法；
- 已经形成具体计划并具备实施资源；
- 有明确实施意向或执行决心；
- 可能出现安排后事、写遗书、交代物品、设定实施时间等行为；
- 但尚未启动自杀行为，或尚未进入正在尝试/正准备尝试阶段。

输出重点：

- 标注为高危线索；
- 清楚列出计划、资源和实施意向依据；
- 说明材料未显示已经开始尝试；
- 提醒咨询师立即按机构危机流程进一步评估和处置。

### 五级：主动式自杀意念，有计划有资源，有实施意向，有尝试或正准备尝试

Sheet2 标准：

```text
主动式自杀意念，有计划有资源，有实施意向，有尝试/正准备尝试
```

Sheet2 定义：

```text
启动执行自杀计划的行为
```

辅助识别线索：

- 来访者已具备主动自杀意念、具体计划、资源和实施意向；
- 已经启动实施行为，或处于即将实施/正在准备实施状态；
- 可能已经服药、打开药瓶、封窗、靠近实施地点、持有工具并即将行动，或明确拒绝救助。

输出重点：

- 标注为极高危线索；
- 说明已经出现尝试、启动实施或正准备尝试的材料依据；
- 提醒必须按机构危机流程、紧急医疗/精神科评估或当地紧急资源处理；
- Agent 不得自行承诺处置或替代现场判断。

## 判定维度

危机等级辅助判断应至少检查四个维度：

| 维度 | 说明 |
|---|---|
| 主动式自杀意念 | 是否存在主动结束生命的意愿，而不只是希望意外死亡或不再醒来 |
| 计划与资源 | 是否存在方式、时间、地点、工具、药物、场景等可执行条件 |
| 实施意向 | 是否已经形成执行决心，是否有遗书、安排后事、明确拒绝救助等线索 |
| 尝试或正准备尝试 | 是否已经启动行为，或正在接近实施动作 |
